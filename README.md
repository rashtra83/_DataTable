# DataTable Project

Responsive Data Table with Searching, Sorting, Pagination using ReactJs.

## Live Demo

https://trial2-c3d0c.web.app/

## Features

The table has following features 
* Sorting on Columns (Name and Email column)
* Filter data using search
* Pagination
